# 开源声明

## 当前状态

AI Agents Leader 正在进行开源准备。

当前仓库会逐步补齐以下内容，确保后续公开后可以稳定协作：

- 文档与命令说明
- 贡献流程
- Issue / Pull Request 规范
- 许可证与发布信息
- 隐私与数据边界说明

在正式公开发布前，仓库内容、目录结构、命令入口和适配器能力仍可能继续调整。

## 项目目标

AI Agents Leader 的目标不是做一个新的 AI IDE、聊天客户端或后台面板。

它是一个本地运行的 AI Runtime Signal System，用尽可能低认知负担的方式告诉你：

- 哪个 agent 正在运行
- 哪个 agent 在等你确认
- 哪个 agent 已完成
- 哪个 agent 出错或可能卡住

核心原则只有一条：看灯就知道 AI 是否还活着。

## 开源后的协作原则

项目正式开源后，会优先接受以下方向的贡献：

- 适配器稳定性和状态识别准确性
- 桌面端与 Web 端的一致性体验
- 文档、示例和接入说明完善
- 各平台兼容性和启动可靠性改进
- 新 agent / 新主题 / 新集成方式的扩展

以下类型的改动通常不会优先接受，除非先有明确讨论：

- 把产品方向改成通用 Dashboard
- 引入与核心目标无关的大型功能模块
- 破坏现有命令语义和状态语义的重构
- 缺少验证与文档说明的大范围结构改动

## 许可证说明

项目计划以 MIT 许可证对外发布。

在正式开源前，以仓库根目录后续提供的 LICENSE 文件为准；本声明用于说明开源方向与协作边界，不单独构成许可证文本。
